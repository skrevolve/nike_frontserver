기본규칙
1. 자기 업무에 따라 이슈하나 생성 (ex. 로그인 페이지 생성)
2. 해당이슈에 맞게 작업을 하면서 commit 메세지를 "[#이슈번호] 작업내용" 로 통일되게 작성
3. commit 을 자기 branch영역에 push가 완료되고 작업이 끝났으면 pull request요청.(pull request요청 승인 대상은 본인제외 서규,주연만 선택)
4. 함부로 merge하짐사ㅔ요
